# Mythfinder (VER 1.2)

# Как скачать аддон? How to download the addon?
- C правой стороны зелёная кнопка с надписью CODE / On the right side there is a green button with the inscription CODE.
- Клик на неё и выбрать Download ZIP / Click on it and select Download ZIP.
- После распаковать в папку игры \Interface\AddOns\ и переименовать в Mythfinder / After unzip it to the game folder \Interface\AddOns\ and rename it to Mythfinder.

# **RU** - Особенности:
- Лёгкий способ найти группу для прохождения m+.
- Гибкая настройка поиска. Все подземелья, либо конкретное. Какой-то определённый лвл ключа, либо диапозон ключей.
- Все найденные в чате игры сообщения LFM дублируются в окно аддона.
- Чтобы аддон начал искать вашу группу в окне аддона нужно: 
- выбрать свою роль (TANK,HEAL или DPS), 
- выбрать конкретное подземелье (или All Dungeons - тоесть все подземелья),
- ввести уровень ключа (или диапозон уровней ключей),
- по желанию выбрать свой спек (пригодится для личного сообщения, что повысит шанс что вас пригласят),
- установить галочку в окошке Link Achieve, что линканёт в сообщении групп лидеру вашу самую высокую ачиву за m+.
- PS если вы стеснительный, то можете не выбирать линк ачивы, она не обязательна для старта поиска.
- Галочка в пункте Announce Bosses подскажет когда бой начался с боссом в групп чат и когда закончился (в скобках сколько по времени длился бой).
- Кнопка об аддоне - думаю и так понятно
- Кнопка Reset сбрасывает выбранное подземелье, тем самым останавливая поиск.
- Кнопка Clear очищает чат в окне аддона.
- Кнопка DN возвращается вниз, в начало чата в окне аддона.


# **EN** - Features:
- An easy way to find a group for m+.
- Flexible search settings. All dungeons or a specific one. A specific key level or range of keys.
- All LFM messages found in the game chat are copied to the add-on window.
- For the add-on to search for your group in the add-on window, you need to:
- select your role (TANK, HEAL, or DPS),
- select a specific dungeon (or All Dungeons - meaning all dungeons),
- enter a key level (or range of key levels),
- optionally select your spec (useful for private messages, which will increase your chances of being invited),
- check the Link Achieve box, which will link your highest m+ achievement to the group leader in a message.
- PS: If you're shy, you don't have to select the achievement link; it's not required to start the search.
- The Announce Bosses checkbox will tell you when the boss fight started in group chat and when it ended (with the duration in parentheses).
- About the addon button - I think it's self-explanatory.
- Reset button resets the selected dungeon, thereby stopping the search.
- Clear button clears the chat in the addon window.
- DN button returns to the top of the chat in the addon window.
